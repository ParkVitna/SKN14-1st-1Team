 # 🚀 프로젝트명 및 개요 
 
 > ## 🏅 Easy Compare (가제) !🏅
 > 각 자동차 모델들의 스펙을 한 눈에 비교할 수 있는 서비스 제공. 


 ## 📌 프로젝트 개요
 - **프로젝트명:** Easy Compare
 - **설명:** 각 자동차 모델들의 스펙을 한 눈에 비교할 수 있는 서비스 제공. 
 - **✨ 주요 기능:**
 > - 스펙을 비교할 자동차 조회 및 선택
 > - 선택한 차종간 스펙 비교
